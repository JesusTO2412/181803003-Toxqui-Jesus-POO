HyperText Markup Languaje

1. ¿Qué significan las siglas HTML?

HyperText Markup Languaje (Lenguaje de marcado de hipertexto) es el lenguaje de marcado estándar para crear páginas web.


  • HTML describe la estructura de una página web
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Los elementos HTML están representados por etiquetas
  • Las etiquetas HTML etiquetan piezas de contenido como "encabezado", "párrafo", "tabla", etc.
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página


2. Las partes de las etiquetas de HTML

Las etiquetas HTML son nombres de elementos rodeados de corchetes angulares:



  • Las etiquetas HTML normalmente vienen en pares como por ejemplo <p>Esta es una etiqueta de parrafo</p>
  • La primera etiqueta en un par es la etiqueta de inicio, la segunda etiqueta es la etiqueta final
  • La etiqueta final se escribe como la etiqueta inicial, pero con una barra diagonal insertada antes del nombre de la etiqueta

Atributos en HTML

Un atributo HTML proporciona información adicional sobre un elemento HTML.


  • Todos los elementos HTML pueden tener atributos
  • Los atributos proporcionan información adicional sobre un elemento
  • Los atributos siempre se especifican en la etiqueta de inicio
  • Los atributos generalmente vienen en pares nombre / valor como: nombre = "valor"

7. ¿Cómo hacer una tabla en HTML

Una tabla HTML se define con la etiqueta <table>. Cada fila de la tabla se define con la etiqueta <tr>. Un encabezado de tabla se define con la etiqueta <th>. Por defecto, los encabezados de las tablas están en negrita y centrados. Una tabla de datos / celda se define con la etiqueta <td>. Ademas, la etiqueta <td> Pueden contener todo tipo de elementos HTML; texto, imágenes, listas, otras tablas, etc.


  • Usar el elemento <table> de HTML para definir una tabla
  • Usar el elemento <tr> de HTML para definir una fila de la tabla
  • Usar el elemento <tr> de HTML para definir los datos de una tabla.
  • Usar el elemento <th> de HTML para definir un encabezado de tabla
  • Usar el elemento <caption> de HTML para definir un título de tabla
  • Usar el elemento border propiedad CSS para definir un borde
  • Usar el elemento border-collapse propiedad CSS para contraer los bordes de las celdas
  • Usar el elemento padding propiedad CSS para agregar relleno a las celdas
  • Usar el elemento text-align propiedad CSS para alinear el texto de la celda
  • Usar el elemento border-spacing propiedad CSS para establecer el espacio entre celdas
  • Usar el elemento colspan atributo para hacer que una celda abarque muchas columnas
  • Usar el elemento rowspan atributo para hacer que una celda abarque muchas filas
  • Usar el elemento id atributo para definir de forma exclusiva una tabla

8. ¿Cómo hacer una lista en HTML


Lista Desordenada

Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la etiqueta. <li> Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada:


  • Coffee
  • Tea
  • Milk


Lista Ordenada

Una lista desordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta. <li> Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada:


  1. Coffee
  2. Tea
  3. Milk


Lista de Descripcion

HTML también admite listas de descripción. Una lista de descripción es una lista de términos, con una descripción de cada término. La etiqueta <dl> define la lista de descripción, la etiqueta <dt> define el término (nombre) y la etiqueta <dd> describe cada término.

10. ¿Cómo crear un formulario en HTML?

Se utiliza un formulario HTML para recopilar la entrada del usuario. La entrada del usuario se puede enviar a un servidor para su procesamiento. Un formulario HTML contiene elementos de formulario. Los elementos de formulario son diferentes tipos de elementos de entrada, como: campos de texto, casillas de verificación, botones de opción, botones de envío y más. El elemento <input> es el elemento de forma más importante. El elemento <input> se muestra de varias maneras, según el atributo de tipo.
Acontinuacion se muestran la forma mas basica de crear un formulario:


Datos del estudiante






Los principales atributos de un input son:


El atributo de tipo

El atributo de entrada type="tipo_de_entrada" especifica el tipo de dato que se ingresa a un campo de entrada.


El atributo de identificacion

El atributo de entrada id="Valor_del_id" especifica identificador para un campo de entrada.


El atributo de nombre

El atributo de entrada name="nombre_del_atributo" especifica un nombre para un campo de entrada.


El atributo de valor

El atributo de entrada value="Valor_del_atributo" atributo de entrada especifica un valor inicial para un campo de entrada.


El atributo de solo lectura

El atributo de entrada readonly especifica que un campo de entrada es de solo lectura.


El atributo deshabilitado

El atributo de entrada disabled especifica que se debe deshabilitar un campo de entrada.


El atributo de tamaño

El atributo de entrada size especifica el ancho visible, en caracteres, de un campo de entrada.


El atributo maxlength

El atributo de entrada maxlength especifica la cantidad máxima de caracteres permitidos en un campo de entrada.


Los atributos min y max

El atributo de entrada min y maxespecifican los valores mínimo y máximo para un campo de entrada.


El atributo múltiple

El atributo de entrada mutiple especifica que el usuario puede ingresar más de un valor en un campo de entrada.


El atributo del patrón

El atributo de entrada pattern especifica una expresión regular con la que se verifica el valor del campo de entrada cuando se envía el formulario.


El atributo de marcador de posición

El atributo de entrada placeholder especifica una pista breve que describe el valor esperado de un campo de entrada (un valor de muestra o una breve descripción del formato esperado).


El atributo requerido

El atributo de entrada required especifica que se debe completar un campo de entrada antes de enviar el formulario.


El atributo de paso

El atributo de entrada step especifica los intervalos de números legales para un campo de entrada. Ejemplo: si step = "3", los números legales podrían ser -3, 0, 3, 6, etc.


El atributo de enfoque automático

El atributo de entrada autofocus entrada especifica que un campo de entrada debería enfocarse automáticamente cuando se carga la página.
